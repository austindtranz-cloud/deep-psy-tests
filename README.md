# DEEP PSY TESTS

**Модульный каталог психологических тестов**, встраиваемый в любой конструктор (в том числе Tilda) через единый API / Smart Loader. Текущая стабильная версия архитектуры: **v11.8**.

## Главные фичи
- **Отсутствие серверной части (Serverless)** — весь движок, квиз, логика подсчета и рендеринга полностью написана на клиентском JavaScript (Vanilla JS).
- **Собственные стили** — независимая дизайн-система Dark/Champagne, спроектированная так, чтобы перекрывать стили CMS (Tilda) и выглядеть дорого.
- **Локальное сохранение прогресса** — результаты и промежуточные стадии квиза сохраняются в `localStorage`, что позволяет продолжить прохождение с любой страницы или отправить группу результатов разом.
- **Smart Cache-Busting Loader (v11)** — лоадер, который сам считывает последний коммит из репозитория GitHub через API и подставляет его как cache-buster `?v=`. Код обновляется у всех клиентов автоматически за доли секунды после PUSH, никаких ручных правок в CMS больше не требуется.
- **Сложная валидация и Scoring** — поддержка простых суммативных тестов (Generic) и сложных диагностических методик вроде СМИЛ (MMPI-2) с K-коррекцией, профильными графиками T-баллов и сложными матрицами ключей (Custom).
- **Плавный UI/UX** — прелоадер для устранения layout-shifts (скачков вёрстки) при подгрузке тяжелых JS; задержки при клике на checkbox для визуальной отдачи (150мс); бесшовная интеграция NumPad клавиатуры (1-9) без конфликтов с инпутами текстовых полей; функция самоочистки бокового меню на внутренних страницах.

## Структура коротко
- `core/` — движок квизов (`engine.js`), рендеринг графиков (`engine_results.js`), боковая панель с историей (`sidebar.js`), модальные окна, дизайн (`style.css`).
- `data/` — базы данных опросников. `test_registry.js` (общий реестр-каталог), `smil_566.js`, `leongard.js` (объемные файлы с десятками/сотнями вопросов и ключами).
- `Алиас/` — файлы-вставки. `js.txt` (лоадер) — вставляется на сайт один раз.
