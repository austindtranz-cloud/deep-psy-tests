# DEEP PSY TESTS

**Модульный каталог психологических тестов**, встраиваемый в Web / CMS (Tilda / Webflow и тд.) через единый API (Smart Loader). 
Текущая стабильная версия архитектуры: **v12.0 STABLE RELEASE**. 

## Особенности (v12.0)
- **100% Serverless** — движок квиза, агрегация результатов, графики (T-баллы) и роутер вычисляются клиентским JavaScript (Vanilla JS).
- **Smart Loader (v11)** — лоадер опрашивает API GitHub на предмет новых коммитов и сбрасывает кэш у клиентов автоматически (`?v=sha`). Отпадает нужда вручную обновлять параметры в CMS.
- **Smart URL-роутер (v12)** — движок читает `window.location.pathname` и самостоятельно подставляет базу тестов. Больше никаких хардкоженных разрозненных шаблонов. Один HTML блок для всего сайта.
- **On-Screen Отладчик** — если категория не найдена, красная ошибка объяснит, какой URL ожидает увидеть роутер.
- **UX Кинезиология** — виртуальные клики на клавиатуре (NumPad), подсветки вариантов, автопереходы, мягкая типографика без использования внешних библиотек-костылей типа SmoothScroll (которые сжигают CPU).
