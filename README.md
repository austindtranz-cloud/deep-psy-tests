# DEEP PSY TESTS — Modular Engine v15.1

**Профессиональная психометрическая система**, оптимизированная для встраивания в Tilda / Webflow. Обеспечивает премиальный пользовательский опыт (UX) и высокую производительность.

Текущая версия: **v15.1 (Modular Design System v4)**.

## Новое в версии v15.1 (Refactoring Update)
- **Design System v4**: Полный переход на CSS-токены (`:root`), шрифты Outfit/Inter и Glassmorphism.
- **Unified UI Templates**: Вся верстка вынесена из JS-логики в объект `DEEP_UI.TEMPLATES` (ui_controller.js).
- **Lazy Loading (Phase 3)**: Динамическая подгрузка вопросов только при запуске теста. Значительная экономия трафика и ускорение LCP.
- **Self-Healing Sidebar**: Боковая панель истории теперь полностью управляется через CSS, очищая старые инъекции стилей.
- **Safari Mobile Fix**: Использование `100dvh` и блокировка эластичного скролла Safari для стабильности оверлея.

## Особенности архитектуры
- **Serverless**: Все расчеты (MMPI-2, Леонгард и др.) выполняются на стороне клиента (Vanilla JS).
- **Smart Loader**: Автоматический сброс кэша через GitHub SHA и строгий порядок инъекции зависимостей.
- **Smart URL Router**: Автоматическое определение категории и визуальной темы по URL страницы.
