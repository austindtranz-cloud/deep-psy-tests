Rules:

- On /tests show all top-level categories, alphabetic library by Russian test names, and keyword library.
- On /tests/personality show all subcategories and all named tests inside them as separate cards.
- Each test card must include title, what it measures, approximate number of items/time, legal/licensing status, and CTA.
- Do web research for every named test and classify: public domain / downloadable with restrictions / academic-only / commercial/proprietary / unknown.
- If a test is proprietary or legally unclear, do not automatically embed verbatim items in the public runnable demo. Use informational card or open replacement.
- Preserve DEEP dark premium style, background #111111, accent #E8D6B3, Tilda-ready code, and paddings 40 / 40 / 20 / 10 px by breakpoint.

Representative legal examples:

- MMPI-2 / СМИЛ -> proprietary / restricted -> Показывать как карточку и route node; не публиковать items verbatim -> Нужна отдельная правовая проверка и, вероятно, лицензирование
- 16PF -> proprietary / restricted -> Карточка + описание; для public runnable demo заменить -> Замену подбирать из open alternatives
- NEO-PI-R / NEO-FFI -> official versions restricted -> Для public demo prefer IPIP-NEO or BFI-2 style route -> Не смешивать official NEO with open analogs без маркировки
- MBTI official forms -> proprietary / restricted -> Не делать 'официальный MBTI' runnable public test -> Можно показать informational card и open Jung-type analog
- IPIP / IPIP-NEO -> public domain -> Можно делать runnable -> Разрешено копировать/редактировать/переводить
- DASS-21 -> download/copy allowed, no modification/profit resale -> Можно делать runnable при соблюдении ограничений -> Не модифицировать wording
- Rosenberg Self-Esteem Scale -> public domain -> Можно делать runnable -> Подходит как открытый якорный тест
- HEXACO -> license for non-academic use -> Карточка + legal note; runnable only after rights check -> Для non-academic use есть royalty/license fee
