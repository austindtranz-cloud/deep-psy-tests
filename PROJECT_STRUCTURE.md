## 1. Что это

**DEEP PSY TESTS** — модульный каталог психологических тестов, встроенный в сайт на Тильде ([deeppsysolutions.com](https://deeppsysolutions.com)). Код хостится на **GitHub Pages**. 
Ключевая особенность v12.0 — полностью автономный **Smart Cache-Busting Loader**, который сам ходит в API GitHub, узнаёт хэш последнего коммита и сбрасывает кэш скриптов, а также **система автоопределения категорий по URL**.

## 3. Инновации и UX (v12.0)

1. **Smart URL Auto-Detection**: Движок больше не требует жёстких `data-category-id` атрибутов. Он самостоятельно анализирует URL (например, `/tests/personality`), ищет в нём английское ключевое слово категории и загружает нужную базу. 
2. **On-Screen Visual Error Handler**: Если URL не совпал (например, страница открыта в предпросмотре Тильды, где адрес `/page123456.html`), скрипт выводит огромный красный блок с текстом о том, что URL некорректен, и даёт инструкцию по исправлению.
3. **Плавный авто-переход (150мс)**: Клик по чекбоксу генерирует визуальную обратную связь (.is-selected), ждёт 150мс и мягко переводит на следующий вопрос.
4. **Умная Клавиатура**: Нажатия `1-9` генерируют синтетический `.click()` по инпуту, полностью проигрывая все анимации, как от мышки.
5. **Self-Healing Sidebar**: Сайдбар автоматически удаляет старые конфликтующие HTML-верстки на страницах и рисует собственный унифицированный чистый DOM.

## 4. Как поднять новую страницу в CMS

Все страницы работают на одном-единственном универсальном куске кода (в T123 блоке), который вообще не нужно модифицировать. 

Главное правило — URL в Тильде ОБЯЗАН заканчиваться на правильное слово: `personality | mental | adaptation | psychiatry | relationships | career | team | organization | psychoanalytic | therapy`.

```html
<!-- Контент категории + роутер -->
<div class="deep-page-container">
  <nav class="deep-nav"><a href="/tests">Все категории</a></nav>
  <div id="deep-categories-container"></div>
</div>

<!-- Модальное окно тестов -->
<div class="deep-tests-overlay" id="deep-tests-overlay" aria-hidden="true">
  <div class="deep-tests-modal">
    <button class="deep-tests-close" aria-label="Закрыть">×</button>
    <div class="deep-tests-app" id="deep-tests-app"></div>
  </div>
</div>
```

## 5. Как обновлять код (Деплой)

Система **serverless** и управляется через Smart Loader.
1. Внеси локальные изменения (добавь тест, поменяй CSS).
2. `git add -A && git commit -m "Update" && git push origin main`
3. Спустя 30-40 секунд (GitHub Pages) изменения автоматически отобразятся у всех пользователей благодаря кэшбастеру! Тильду трогать больше не нужно.
