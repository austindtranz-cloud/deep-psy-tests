# 🛠 Доступные навыки (Skills) Antigravity

Ниже представлен полный список встроенных навыков ИИ-ассистента и описание того, что они позволяют делать. Эти навыки активируются автоматически, когда запрос попадает под их специализацию.

---

### 🎨 Дизайн и Графика
* **UI/UX Pro Max (ui-ux-pro-max)** — Продвинутая база знаний по UI/UX-дизайну для веба и мобильных платформ. Помогает с выбором стилей (Glassmorphism, Dark mode и др.), шрифтов, цветовых палитр, а также проводит аудит (accessibility, contrast, layout) и дает рекомендации по компонентам (модальные окна, списки, формы) для React/Next.js/React Native и т.д.
* **Canvas Design (canvas-design)** — Создание красивого визуального искусства, постеров и дизайнов (в виде PNG или PDF-документов) с применением правильной дизайнерской философии.
* **Algorithmic Art (algorithmic-art)** — Создание алгоритмического и генеративного искусства с помощью библиотеки `p5.js` (частицы, потоковые поля).
* **Theme Factory (theme-factory)** — Инструментарий для применения заранее настроенных стилей и тем оформления к различным артефактам (слайдам, отчетам, HTML-лендингам). Помогает быстро сделать результат стильным.
* **Brand Guidelines (brand-guidelines)** — Навык по применению фирменных цветов и типографики (на основе брендбука Anthropic) к документам и дизайну.

### 📄 Работа с Офисными Документами
* **Word (docx)** — Полный спектр работы с документами Microsoft Word. Ассистент может создавать, читать, форматировать профессиональные документы, вставлять изображения, добавлять оглавление и извлекать текст из файлов `.docx`.
* **Excel / Таблицы (xlsx)** — Глубокая работа с таблицами (`.xlsx`, `.csv`, `.tsv`). Включает парсинг, создание файлов с нуля, очистку "грязных" данных, расчеты формулами, добавление новых столбцов.
* **PowerPoint (pptx)** — Работа с презентациями. ИИ может сгенерировать слайды, прочитать существующий файл `.pptx` (чтобы достать оттуда текст для пересказа), разбивать/объединять файлы и работать с макетами.
* **PDF (pdf)** — Мощный инструмент работы с PDF: чтение текста и таблиц, объединение/разделение файлов, поворот страниц, добавление водяных знаков, шифрование и даже OCR (распознавание текста) на отсканированных картинках в PDF.

### 📝 Тексты и Документация
* **Doc Co-authoring (doc-coauthoring)** — Структурированный помощник по совместному написанию технической документации, проектных предложений и спецификаций (RFC/Specs). Помогает собрать контекст, улучшить черновики и сделать текст читабельным.
* **Internal Comms (internal-comms)** — Помощь в написании внутренних корпоративных коммуникаций: отчетов о проделанной работе, обновлений для руководства, сводок об инцидентах и FAQ.

### 💻 Разработка и Инфраструктура
* **Web Artifacts Builder (web-artifacts-builder)** — Инструмент для быстрой сборки сложных и интерактивных компонентов прямо в интерфейсе с использованием современного стека: React, Tailwind CSS и библиотеки `shadcn/ui`.
* **Testing (webapp-testing)** — Инструментарий для автоматизированного E2E тестирования локальных веб-приложений с помощью *Playwright*. ИИ может управлять браузером, проверять работу фронтенда, делать скриншоты и отлаживать логику UI.
* **Claude API (claude-api)** — Помощник по созданию приложений, интегрирующих модель Claude через Anthropic SDK. Знает последние стандарты и фишки работы с API.
* **MCP Builder (mcp-builder)** — Руководство по созданию высококачественных серверов MCP (Model Context Protocol), которые связывают ИИ с внешними сервисами и базами данных. Поддерживает разработку на Python и TypeScript.
* **Skill Creator (skill-creator)** — Мета-навык для создания новых навыков, их доработки, оптимизации и бенчмаркинга ИИ на основе этих навыков.

### 💡 Прочие утилиты
* **Slack GIF Creator (slack-gif-creator)** — Создание анимированных GIF-изображений, специльные инструменты валидации и анимации для использования в мессенджере Slack.
